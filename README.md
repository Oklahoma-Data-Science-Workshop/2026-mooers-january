# Using Obsidian with AI to support scientific writing

## Supplemental materials
The repository contains the supplemental materials for the ODSW talk given on 2026 January 16 by Blaine Mooers.

1. mooers2026jan16.pdf: slides shown.
2. ODSWTutorial2026.zip: tutorial Vault for use with Obsidian. 




## Do the following before the meeting (will take 5-15 minutes)

1. Download [Obsidian](https://obsidian.md/) for your operating system and install by drag and drop.
2. Download the zip file ODSWTutorial2026.zip:
   - Click on ODSWTutorial2026.zip above.
   - Select "Download raw file" button in the upper righthand corner.
3. Unzip the zip file in your home directory.
4. Start Obisidian by double clicking on its icon.
5. From the toolbar or Obsidian GUI, select File --> open vault (triggers a GUI to open).
6. Select "Open folder as vault"
7. Navigate to ODSWTutorial2026 and select it. A list of >30 files will appear.

## For developers

The zip file contains the homemade plugin bidirectional-folgezettel.
The Typscript source file and tests are located [here](https://github.com/MooersLab/bidirectional-folgezettel).
You are welcome to make pull requests.
